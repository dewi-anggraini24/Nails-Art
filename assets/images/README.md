## Image Optimization

### Before Upload
1. **Resize**: Use correct dimensions
2. **Compress**: Use tools like:
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - ImageOptim (Mac)
3. **Format**: 
   - Use JPG for photos
   - Use WebP for better compression (modern browsers)
   - PNG only if transparency needed

### Compression Tips
- Target file size: 100-300KB per image
- Quality: 80-85% is usually perfect
- Remove EXIF data for privacy
